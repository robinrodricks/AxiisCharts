Classpublic class VerticalBoxLayout
InheritanceVerticalBoxLayout AbstractLayout flash.events.EventDispatcher
Implements mx.binding.IBindingClient

The VerticalBoxLayout is an AbstractLayout that creates a RegularRectangle for each item in its dataProvider. The RegularRectangles are sized and spaced evenly across the height of the VerticalBoxLayout.

Classpublic class HAxis
InheritanceHAxis AxisBase BaseLayout AbstractLayout flash.events.EventDispatcher
Implements mx.binding.IBindingClient

HAxis renders values from left to right between a minimum and maximum value, and is intended to be rendered beneath other layouts to provide a point of reference for those visualizations.

HAxis can optionally label each major tick, render gridlines aligned to the axis itself.

The majority of the functionality of HAxis is contained within its super class, AxisBase.
